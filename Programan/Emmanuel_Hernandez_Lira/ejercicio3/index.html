Instrucciones

Genera una interfaz donde haya 3 inputs y un botón buscar.

Valida que los 3 inputs estén completos y sean numéricos al momento de presionar buscar.

Busca el pokemon del ID del primer input recién cuando finalice la búsqueda del segundo. Haz lo mismo con el tercero.

Lanza cada una de las llamadas cuando estés seguro que la anterior finalizo.
